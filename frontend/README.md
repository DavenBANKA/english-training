# Frontend - Plateforme de Test

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le frontend sera accessible sur http://localhost:3000

## Build

```bash
npm run build
```

## Structure

- `/src/pages` - Pages de l'application
- `/src/components` - Composants réutilisables
- `/public/images` - Images statiques

## Design

- Couleurs principales: Orange (#ff6b35) et Blanc
- Design professionnel et moderne
- Navigation fixe avec cadre rectangulaire
- Section hero avec image de fond stylisée
